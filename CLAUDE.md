## Project

One-page Next.js website — countdown to a flight to Japan.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS

## Design & Theme

- Japanese aesthetic: sakura blossoms, Japanese flag motifs
- Light background, pink/white sakura tones, red accents (Japanese flag), dark gray/black text
- Modern, clean layout

## Page Structure

1. **Countdown section** — days, hours, minutes remaining until the flight
2. **Boarding pass section** — styled like a real airline ticket

## Flight Details

- Route: Belgrade (BEG) → Tokyo Narita (NRT)
- Date: Wednesday, 11 November 2026
- Departure: 20:15 BEG
- Arrival: 19:10 NRT (next day)
- Duration: 14h 55m
- Connection: Istanbul (IST)
- Flights: TK1084 (BEG → IST), TK0300 (IST → NRT)
- Class: Economy (V)
- Airline: Turkish Airlines (use official Turkish Airlines logo)

## Interactivity

- Falling sakura petals animation (CSS, full screen background)
- Flip/split-flap animation on countdown numbers (like old airport departure boards)
- Boarding pass hover effect (subtle lift + glow)
- Cursor trail of sakura petals
